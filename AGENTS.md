# Agent Guidelines

## Documentation

When making any changes to this repo, update `README.md` to reflect those changes:

- **New script added** — add a section under `## Usage` with a description, usage examples, and any dependencies.
- **Script modified** — update its existing section if behaviour, flags, or dependencies changed.
- **Script removed** — remove its section from the README.
- **`install.sh` changed** — update the `## Installation` section if the install behaviour changed.

Keep README sections consistent with the existing format: a heading with the script name, a short description, a code block showing usage, and a dependencies line if applicable.
